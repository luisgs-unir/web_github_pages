Crear un Repositorio en GitHub

  1. Visita GitHub.com y crea una cuenta si no tienes una.
  2. Haz clic en el botón "Nuevo repositorio" en la esquina superior derecha.
  3. Asigna un nombre a tu repositorio. Si es un sitio de usuario, debe ser <usuario>.github.io.
  4. Selecciona la visibilidad del repositorio (público o privado).
  5. Inicializa el repositorio con un README si lo deseas.
  6. Haz clic en "Crear repositorio".

Subir Archivos a tu Repositorio


git clone https://github.com/usuario/nombre-repositorio.git
cd nombre-repositorio
echo '<html><body>Hola Mundo</body></html>' > index.html
git add .
git commit -m "Agregar archivo index.html"
git push origin main

Configurar GitHub Pages

  1. Ve a la página de tu repositorio en GitHub.
  2. Haz clic en la pestaña "Configuración" (Settings).
  3. Desplázate hacia abajo hasta la sección "GitHub Pages".
  4. En "Source", selecciona la rama que deseas usar (por ejemplo, main o gh-pages) y haz clic en "Guardar".

Visitar tu Sitio Web

Una vez que GitHub Pages esté configurado, tu sitio estará disponible en la URL:


https://<usuario>.github.io/<nombre-repositorio>/

Crear un sitio de GitHub Pages para un repositorio concreto

Como acabamos de ver, cada usuario de GitHub puede tener un sitio de GitHub Pages para su cuenta personal y también puede crear sitios de GitHub Pages para cada uno de sus repositorios. Lo que nos permite tener múltiples sitios bajo la misma cuenta de usuario.

Para ello sólo tenemos que:

  1. Acceder al repositorio correspondiente.
  2. Entrar en la configuración del repositorio "Settings"
  3. En la sección "Pages"
  4. Configuramos el reposito:
    1. Source: Deploy from a branch
    2. Branch: Configuramos la rama y la ruta en la que tenemos el sitio web creado/almacenado
    3. Save pulsamos el botón para guardar los cambios
